# DecodeLabs Week 1 - Iris Flower Dataset Analysis

**Project Completed as part of DecodeLabs Internship Program**

---

## 📋 Project Overview

This project demonstrates a complete **End-to-End Data Analysis Pipeline** on the classic **Iris Flower Dataset**. It covers data collection, cleaning, and exploratory data analysis (EDA) using Python.

### 🎯 Objectives
- Load and understand the dataset
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis with visualizations
- Extract meaningful insights about the three iris species

## 📊 Key Insights

1. **Setosa** has significantly smaller petals compared to the other two species.
2. **Petal Length** and **Petal Width** are highly correlated (r ≈ 0.96).
3. **Virginica** generally has the largest petals.
4. Sepal width shows the most variation and outliers across species.
